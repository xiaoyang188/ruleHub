# 先自检，再提交

广告网络会评估站点是否对访客有价值、信息是否透明、体验是否正常。下面是一份偏实操的体检表，可对照 RuleHub 类内容+工具站使用。

# 内容

- 是否有一定数量的**原创**深度页（教程/指南/评测），而非仅有聚合列表
- 文章是否围绕站点主题，形成专栏（如 [/insights](/insights)）
- 是否避免复制、隐藏文字、误导点击

内容策略参见 [SEO 与内容站](/insights/seo-for-ai-content-sites) 与 [内容产品联动](/insights/content-product-flywheel)。

# 透明与法律

- 关于我们、隐私政策、服务条款、联系方式可从页脚到达
- 隐私政策说明 Cookie / 广告合作方（若适用）
- 联系邮箱真实可收信

# 体验与技术

- HTTPS 正常，移动端可读
- 导航一致，关键栏目可发现
- 404/空状态不至于让人以为站点废弃
- 无整页弹窗阻断阅读（可接受的公告除外）

# 行为与流量

- Search Console 已验证，sitemap 已提交
- 有真实访问与收录，而不是纯空壳
- 不买垃圾流量、不参与点击欺诈

# RuleHub 当前建议动作

1. 部署最新专栏与法律页
2. GSC 检查 `/insights/*` 覆盖
3. 抽查联系表单/邮箱
4. 再提交广告申请，并继续保持更新频率

# 小结

可信度是内容和透明信息的合力。清单不能保证通过，但能避免「明显没准备好」的提交。
